# EEG Robotic Hand using Raspberry Pi

## Overview
This project is an EEG-based robotic hand developed using Raspberry Pi, Python, and NPG Lite for real-time signal processing and robotic finger actuation.

## Features
- Real-time EEG signal monitoring
- Threshold-based gesture classification
- Multi-servo finger control
- GPIO-based PWM signal control
- Signal calibration and noise filtering

## Components Used
- Raspberry Pi
- NPG Lite EEG Module
- Servo Motors
- Jumper Wires
- Power Supply

## Technologies Used
- Python
- GPIO PWM
- Serial Monitoring

## Working
The EEG signals are acquired using the NPG Lite module and processed in Raspberry Pi using Python. Based on threshold values, the robotic fingers are actuated using PWM signals.
